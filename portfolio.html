Welcome to My System Development Portfolio

As a dedicated system developer student, I'm excited to showcase the culmination of my learning journey through three impactful projects: an ATM machine, a bank application, and a comprehensive database system for a school. You can also visit my GitHub to see all of my projects.

    ATM

    • The program simulates a banking system with a console interface. It's object-oriented, utilizing classes and lists for accounts and users. Users log in with a username and PIN, accessing options like viewing balances, transferring money, and withdrawing funds. The code structure aims for readability and modularity, with custom methods for menu navigation and transactions. Reflection notes the transition from arrays to object-oriented design for improved organization and readability.

    Sharp Shooters Bank

    • Sharp_Shooters is a C# banking system designed for users to manage accounts, perform transactions, and borrow money securely. It offers a user-friendly interface with features for both regular customers and administrators. The system includes classes for managing accounts, handling administrative tasks, providing banking functionalities, managing currency exchange, initiating transfers, defining user properties, and improving the user interface. Users can open regular and savings accounts, perform administrative tasks, log in securely, manage currency exchange rates, calculate loan limits, perform currency conversions, and access utility functions for a better user experience.

    School Database

    • This C# project is a school database management system designed to facilitate the management of employees, students, and other administrative tasks within a school environment. It includes functionalities for handling database migrations, defining entity models for the database schema, managing employees and students, as well as utility methods used across the project. The main entry point of the application is the Program.cs file.
